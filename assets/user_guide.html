Introduction

ISCO is an advanced framework designed for the automated analysis of organoid microscopy images. It combines state-of-the-art image segmentation with comprehensive analysis tools, providing researchers with an efficient solution for organoid characterization.

Getting Started

System Requirements

  • Operating System: Windows 10 or later
  • RAM: 8GB minimum (16GB recommended)
  • GPU: NVIDIA GPU with CUDA support (recommended)
  • Storage: 2GB free space for installation

Installation

  1. Download the ISCO installer from the official website
  2. Run the installer and follow the setup wizard
  3. Launch ISCO from the desktop shortcut or start menu

Model Selection Guide

ISCO provides four different model sizes to accommodate various needs and hardware capabilities:

Model Size Memory Usage Processing Speed Accuracy Recommended Use Case
Tiny ~2GB VRAM Fastest Good Quick analysis, limited hardware resources
Small ~4GB VRAM Fast Better Balance between speed and accuracy
Base ~8GB VRAM Moderate Excellent High-quality analysis, standard hardware
Large ~12GB VRAM Slowest Best Maximum accuracy, high-end hardware
Note: Memory usage is approximate and may vary based on image size and system configuration.
Recommendation: Start with the Base model for most use cases. If you need faster processing, switch to Small or Tiny. For maximum accuracy and if you have sufficient hardware resources, use the Large model.

Workflow Guide

1. Data Preparation

  • Prepare your microscopy images in PNG format
  • Optional: Create a mapping file (CSV) for group analysis
  • Ensure sufficient disk space for results

2. Analysis Setup

  1. Select your preferred model size based on your hardware capabilities
  2. Choose the input directory containing your images
  3. Specify an output directory for results
  4. Optional: Load a mapping file for group analysis

3. Processing and Results

  • Monitor progress through the progress bar
  • Review generated masks and measurements
  • Use the merge function to combine results

Analysis Metrics

ISCO provides comprehensive morphological analysis including:

  • Area: Total surface area of the organoid
  • Perimeter: Boundary length measurement
  • Radius: Average distance from center to boundary
  • Non-smoothness: Boundary roughness indicator
  • Non-circularity: Deviation from perfect circular shape
  • Symmetry: Measure of shape regularity

Troubleshooting

Common Issues:

  • Model Loading Error: Verify model files are in the correct directory
  • Processing Speed: Consider using a smaller model size or enabling GPU acceleration
  • Memory Issues: Close other applications or switch to a smaller model size
  • Image Loading Error: Ensure images are in PNG format and not corrupted
